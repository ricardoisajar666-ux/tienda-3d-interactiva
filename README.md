# Tienda 3D Interactiva

Una tienda 3D completa donde puedes caminar, saltar y explorar.

## Caracteristicas

- **Movimiento FPS**: Camina con W/A/S/D
- **Saltar**: Barra espaciadora
- **Correr**: Shift izquierdo
- **Mirar**: Mouse
- **Interactuar**: Click en productos
- **Avatar interactivo**: Personaje animado con expresiones
- **Productos 3D**: Tartas, cupcakes, macarons, donuts, empanadas
- **Iluminacion profesional**: 5 spots + point lights
- **Efectos visuales**: Bloom, vignette, sombras

## Controles

| Tecla | Accion |
|-------|--------|
| W/A/S/D | Caminar |
| ESPACIO | Saltar |
| SHIFT | Correr |
| MOUSE | Mirar |
| CLICK | Ver info del producto |

## Como Ejecutar

1. Abrir terminal en esta carpeta
2. Ejecutar: `python server.py`
3. Abrir http://localhost:8080
4. Haz clic para empezar

## Tecnologias

- Three.js v0.160.0
- WebGL 2.0
- GLTFLoader
- EffectComposer (Bloom, Vignette)

## Modelos 3D

- Avatar: Pastel con cara expresiva
- Productos: 5 tipos de postres
- GLB: Modelo Salpicon (carga dinamica)
